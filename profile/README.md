# FuckSchools

## Core Thesis
FuckSchools exists to replace content-first education with demand-driven building. The platform does not teach. It listens for intent, determines the smallest useful next step, and routes the learner toward a concrete build path.

## Axioms
- The platform does not teach.
- It provides plans and routes for building.
- Motivation comes before content inventory.
- Knowledge transfer between people costs.
- Legibility matters more than ceremony.
- Hard blocks are recursively materialized.
- Traversal is demand-driven.
- Only the next frontier matters.
